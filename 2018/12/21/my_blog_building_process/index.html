工具

  • git
  • github账号
  • node.js
  • hexo

因为每个github账号都可以开一个自己的静态网站(github pages服务),简单还免费,只需要创建一个名称为username.github.io的repository。(username为你的账号名称)这个repository将储存你的blog的文件

Git是一个版本工具，此处用来将你的修改推送到你的github上(需要使用SSh赋予你的电脑推送东西到github的权限)

hexo博客生成工具，为什么用它，彬彬推荐的，我不懂hhh。它可以自动生成html+css+js，你不需要再去构造你blog的形状。
